  • getTermOfUse(scooterOrPackageId) - Kullanım koşullarını getirir
  • getSaleContract(scooterOrPackageId) - Satış sözleşmesini getirir
  • getCommercialAgreement(country) - Ticari sözleşmeyi getirir
  • getContractArm(lang) - ARM sözleşmesini getirir

Varsayılan ülke listesini getirir. Şu anda API çağrısı yerine statik veri kullanılmaktadır.

Dönüş Değeri

Future<List<Country>>

CountryModel 

{
    "bayrakURL": "https://example.com/flags/tr.png",
    "ulkeAdiTurkce": "Türkiye",
    "ulkeAdi": "Turkey",
    "telefonKodu": 90,
    "ulkeKodu2Hane": "TR",
    "ulkeKodu3Hane": "TUR",
    "telefonNoFormat": "(###) ### ## ##"
}
Not
API çağrısı şu anda devre dışı bırakılmıştır. Bunun yerine defaultValuesCountries sabit verisi kullanılmaktadır. Bu, 500 veya benzeri sunucu hatalarında bile ülke bayraklarının gösterilebilmesini sağlar.
Örnek Kullanım
Dart 

final countryApi = CountryApi();
final countries = await countryApi.getCountries();

// Ülke listesini kullanma
for (var country in countries) {
    print(country.name);
    print(country.flag);
}
